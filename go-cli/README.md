# jikan-rest-cli

AQL-driven CLI and REPL for the JikanRest SDK. Positional arguments are
joined into a single AQL expression and evaluated; with no arguments,
falls into an interactive REPL.

Built on `github.com/aql-lang/aql/eng/go` and the sibling Go SDK at `../go`.

## Build

```sh
go build -o jikan-rest-cli ./...
```

## Run

```sh
# One-shot: arguments form a single AQL expression
./jikan-rest-cli list anime
./jikan-rest-cli load 1 anime
./jikan-rest-cli load '{id:1}' anime

# REPL
./jikan-rest-cli
```

## Words

| Word     | Signatures                                   | Description                |
|----------|----------------------------------------------|----------------------------|
| `list`   | `[entity]` · `[query entity]`                | List records               |
| `load`   | `[entity]` · `[query entity]`                | Load a single record       |
| `update` | `[entity]` · `[query entity]`                | Update a record            |

`query` is either a Map (`{id:1}`) or a Scalar (`1`, treated as `{id:1}`).
`entity` is one of the SDK's entity names (auto-quoted as an atom).

## Entities

anime character club external genre magazine manga people_search person producer random recommendation review schedule season top user user_about user_club user_friend user_history user_statistic user_update watch_episode watch_promo

## REPL commands

- `:quit` / `:q` / `:exit` — exit the REPL
- `:help` / `:h` / `:?`     — show help
